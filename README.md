# Цингеровские Эфемериды (Геодезическая Астрономия)

Программа для вычисления местного звездного времени $s_0$ в момент равенства зенитных расстояний двух звезд (способ Цингера). Позволяет строить точные астрономические эфемериды на основе численных методов.

## Реализованные методы
* **Метод дихотомии (Bisection method)** — используется для поиска глобального корня на отрезке $[0, 2\pi]$ и для безопасного подбора начального приближения.
* **Метод Ньютона (Sir Isaac Newton method)** — используется для сверхбыстрого уточнения рабочего корня с точностью до $10^{-15}$.

> **Важно:** В алгоритмы встроена аппаратная защита от "залипания" итераций на границах машинного нуля типа `double` при делении отрезков.

## Структура проекта
* `struct.h` — описание структуры данных `data` для хранения координат.
* `methods.h` — прототипы численных методов и функций нормализации.
* `main.c` — парсинг аргументов командной строки и запуск вычислений.
